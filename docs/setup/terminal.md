# Windows
1、windows返回上一个应用：Alt + Tab<br>
2、windows中打开/关闭F1~12功能键：Fn + ESC<br>
3、拷贝公司代码打开的终端中粘贴：Shift + Insert<br>
4、开始菜单搜索`PowerShell`（以管理员身份运行）,即可打开`C:\Users\admin`终端<br>
5、鼠标变成方块的恢复方法：按Insert

# IDEA
1、快速创建：<br>
(1)idea中生成无参、有参、get、set方法：Alt + Insert<br>
(2)idea中快速创建一个接口的实现类：Alt + 回车<br>
(3)idea中快速生成局部变量：Alt + Enter<br>
(4)idea中打开命令行窗口：F12或Alt + F12
